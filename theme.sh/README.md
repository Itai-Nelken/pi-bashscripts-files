# theme.sh
https://github.com/lemnos/theme.sh

### Installation
```
sudo apt install fzf -y
git clone https://github.com/lemnos/theme.sh
cd theme.sh
chmod +x theme.sh && sudo cp theme.sh /usr/bin/

# Run theme.sh
theme.sh
```
