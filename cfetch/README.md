# cfetch
 A simple system information tool for Linux. 
 
 ```
 # Git clone the repo then go to the cfetch directory
git clone https://github.com/clieg/cfetch.git

# Compile cfetch
make compile

# Install cfetch
sudo make install

# Run cfetch
cfetch

# Manpage
man cfetch

# Uninstall cfetch
sudo make uninstall
```
